# 项目简介

## 项目名称

UCI Member（优迪邦成员端小程序）

## 项目定位

这是优迪邦智能洗车平台的用户端小程序应用，用于全面替代原有的基于 PHP View 模版渲染的移动端前端。通过现代化的前后端分离架构，提供更流畅、更易维护的用户体验。

## 核心目标

1. **功能迁移**：将原有 `backend/app/uci` 中 PHP View 渲染的所有用户端功能迁移到独立的小程序前端
2. **架构升级**：采用 Vue3 + TypeScript + uni-app 技术栈，实现跨平台支持（微信小程序、支付宝小程序、H5）
3. **体验优化**：提供更快速、更现代的用户界面和交互体验
4. **独立开发**：作为完全独立的仓库进行开发和维护，与后端 API 通过接口交互

## 用户群体

- **主要用户**：优迪邦智能洗车平台的终端消费者
- **使用场景**：扫码洗车、购买优惠券、查询订单、充值消费、推荐返佣等

## 技术选型理由

- **Wot Starter**：成熟的 uni-app 脚手架，提供完整的开发工具链
- **Wot UI**：70+ 高质量组件，支持多端适配和主题定制
- **Vue3 + TypeScript**：提供类型安全和现代化的开发体验
- **Alova**：高效的请求工具，简化 API 调用
- **Pinia**：Vue3 官方推荐的状态管理方案

## 项目边界

### 包含范围

- 所有用户端功能（个人中心、优惠券、充值、订单、推荐等）
- 与后端 API 的交互
- 多端适配和发布

### 不包含范围

- 后端 API 开发（由独立的后端项目提供）
- 管理后台功能
- 硬件设备交互逻辑（由后端处理）

## 成功标准

1. 完全替代原有 PHP View 前端的所有功能
2. 支持微信小程序、支付宝小程序、H5 三端运行
3. 用户体验流畅，响应迅速
4. 代码结构清晰，易于维护和扩展
5. 通过生产环境测试，无重大 Bug

## 关键约束

1. **API 依赖**：必须与现有后端 API 保持兼容
2. **独立性**：不依赖其他项目的文件路径，所有配置和资源都在本项目内
3. **多端兼容**：确保在微信小程序、支付宝小程序、H5 三端都能正常运行
4. **用户体验**：页面加载和交互响应时间要比原有 View 渲染方式更快
