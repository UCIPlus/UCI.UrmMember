## 任务管理说明

本文件用于跟踪项目中的所有任务。每个任务都有独立的文件，格式为 `TASKID-taskname.md`。

### 任务状态
- **Pending**：待开始
- **In Progress**：进行中
- **Completed**：已完成
- **Abandoned**：已放弃

### 如何使用
1. 创建新任务：使用命令 `add task` 或 `create task`
2. 更新任务：使用命令 `update task [ID]`
3. 查看任务：使用命令 `show tasks [filter]`

### 任务示例
```markdown
## Pending
- [TASK001] 实现用户登录功能 - 等待后端 API 文档
- [TASK002] 创建个人中心页面 - 等待设计稿

## In Progress
- [TASK003] 配置 Alova 实例 - 正在对接后端 API

## Completed
- [TASK000] 项目初始化 - 已于 2026年1月8日完成
```
